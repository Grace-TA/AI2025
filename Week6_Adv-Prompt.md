## A. 進階解放ChatGPT實力: 明確指定角色（Role Assignment）

### 案例 1：旅遊行程規劃 

以下是兩個不同的旅遊行程請求所產生的輸出比較，透過表格方式來說明它們的不同點，以便教育性地分析這兩種提示（Prompt）對生成結果的影響。  

| **比較項目** | **Prompt 1：幫我設計一個5天的京都旅遊行程。** | **Prompt 2：你是一位專業的旅遊規劃師，請為家庭旅遊設計一個5天4夜的京都文化旅遊行程，包含每日交通與餐飲建議。** |
|-------------|------------------------------------|---------------------------------------------------------|
| **指令清晰度** | 相對簡單，範圍較寬泛 | 更明確，設定了特定的旅遊目標與條件 |
| **旅遊主題** | 可能涵蓋各類行程（如購物、美食、文化等） | 主要聚焦在 **家庭旅遊** 與 **文化體驗** |
| **住宿資訊** | 可能未特別詳細列出 | 可能包含適合家庭的住宿建議（如和式旅館） |
| **交通方式** | 可能僅列出主要景點 | 明確提供每日交通方式（如地鐵、巴士、步行） |
| **餐飲建議** | 可能未提及或僅簡單列出 | 會針對每一日的行程搭配適合的餐廳或特色美食 |
| **行程規劃細節** | 一般列出每日景點，但細節不多 | 會更詳細，包括參觀時間、適合家庭的活動 |
| **適用族群** | 適合個人、情侶或自由行旅客 | 針對 **家庭旅遊**，考慮小孩或長者的需求 |
| **教育性價值** | 幫助使用者獲得基本旅遊概念 | 提供完整的旅遊規劃技巧，幫助學習如何設計行程 |
| **資訊結構** | 可能是簡單的「景點列表」 | 以 **時間軸** 或 **行程表** 方式組織內容，結構更清晰 |

### **🔍 教育啟示**
1. **Prompt 越具體，AI 產出的內容越符合需求。**  
   - Prompt 1 的結果較為廣泛，而 Prompt 2 加入了「專業旅遊規劃師」、「家庭旅遊」、「文化旅遊」、「交通與餐飲」等要求，使 AI 生成的內容更符合實際需求。  

2. **設定清晰的條件有助於獲得高品質的結果。**  
   - Prompt 2 透過設定「5天4夜」、「包含交通與餐飲」等條件，確保 AI 產出的資訊完整且實用。  

3. **適用對象不同，規劃方式也會不同。**  
   - Prompt 1 適合自由行者，內容可能更靈活。  
   - Prompt 2 則考慮到家庭成員（如小孩、長輩）的需求，使行程更友善。

### 案例 2：職業規劃建議

以下是兩個不同的請求（Prompt）所產生的輸出比較，透過表格方式來分析它們的不同點，並從教育的角度解釋如何優化 Prompt 以獲得更精確的 AI 回應。  

| **比較項目** | **Prompt 1：給我一些進入人工智慧領域的建議。** | **Prompt 2：你是一位專業職場導師，請根據以下背景提供進入人工智慧領域的具體職業規劃建議，包括學習路徑與實習機會。** |
|-------------|------------------------------------|---------------------------------------------------------|
| **指令清晰度** | 指令較為寬泛，範圍較廣 | 指令明確，設定了具體的需求與內容 |
| **適用族群** | 針對所有希望進入 AI 領域的人 | 針對 **特定背景（如半導體、測試技術）** 量身打造 |
| **建議類型** | 一般性的學習方向，如「學 Python」、「學機器學習」 | 具體的職業發展計畫，包括 **學習內容、實習機會、應用場景** |
| **內容結構** | 可能是簡單的建議列表 | 以 **分階段學習**、**產業應用**、**職涯發展** 方式組織內容 |
| **細節深度** | 建議較為表面，如「學習 TensorFlow」或「參加 Kaggle 競賽」 | 提供 **進階學習建議（如 Transformer、邊緣 AI）** 並結合用戶背景分析 |
| **實習與職場機會** | 可能僅列出熱門 AI 公司，如 Google、OpenAI | 針對用戶背景推薦合適的企業與研究機構（如 TSMC AI Lab、NVIDIA Edge AI） |
| **應用領域** | 總體 AI 領域，如計算機視覺、自然語言處理 | 聚焦於 **AI + 半導體測試 / 嵌入式 AI / AIoT 應用** |
| **職業規劃** | 可能未提及或僅給出大方向 | 具體的短期（0-6 個月）、中期（6-12 個月）、長期（1-3 年）職業發展計畫 |
| **教育性價值** | 提供基本方向，但不夠細緻 | 提供 **完整的學習路徑與職業發展策略**，能直接應用於職涯規劃 |

---

### **🔍 教育啟示**
1. **Prompt 具體化，能讓 AI 產生更精確的建議。**  
   - Prompt 1 適合一般人尋求初步 AI 建議，但缺乏個人化指導。  
   - Prompt 2 透過設定「專業職場導師」、「根據背景提供建議」、「包含學習路徑與實習機會」，讓 AI 提供更符合個人需求的職業發展建議。  

2. **提供背景資訊，有助於 AI 生成有針對性的回應。**  
   - 如果你有 **特定專業（如半導體、嵌入式系統）**，請務必提供這些資訊，以獲得更精確的建議。  

3. **結構化輸出更易理解與應用。**  
   - Prompt 1 的輸出可能是零散的學習建議，而 Prompt 2 會以「短期→中期→長期發展」來組織內容，使計畫更可執行。

### 案例 3：語言學習計畫

以下是 **「如何提升我的英語口說能力？」** 和 **「你是一位語言學習教練，請為學生設計一個30天提升英語口說能力的學習計畫，包括每日活動與教材建議。」** 這兩個不同的問題，對應的輸出比較表：  

| **比較項目** | **簡單問題：「如何提升我的英語口說能力？」** | **專業問題：「請設計30天英語口說學習計畫」** |
|------------|---------------------------------|------------------------------------------------|
| **回答範圍** | 提供 **概括性的學習方法與技巧** | 提供 **詳細的學習計畫，包含每日活動與教材** |
| **內容深度** | 屬於 **一般建議**，適合剛開始學習者 | 屬於 **專業級規劃**，適合有明確目標的學習者 |
| **學習方法** | 1. 聽力輸入（Podcast、TED Talks）  <br> 2. 影子跟讀（BBC、ELSA）  <br> 3. 即興對話練習（Tandem、italki）  <br> 4. 錄音回聽（Speechling） | 1. **Week 1**：發音與語感（影子跟讀、朗讀）  <br> 2. **Week 2**：流暢度訓練（即興說話、新聞簡述）  <br> 3. **Week 3**：專業領域表達（職場報告、科技討論）  <br> 4. **Week 4**：實戰應用（線上交流、模擬面試） |
| **學習進度** | **彈性學習**，沒有明確的時間表 | **30 天的逐步計畫**，每天有具體任務 |
| **教材推薦** | 推薦常見學習工具（如 BBC、TED、Podcast） | 每天指定具體教材（如 TED Talks、Lex Fridman Podcast、Google Voice Typing） |
| **適合對象** | **初學者或希望獲得基本建議者** | **有明確學習目標的進階學習者** |
| **應用場景** | 一般日常對話、提升基本流暢度 | 針對專業表達、商務英語、職場溝通 |

### **📌 主要結論**
- **第一個問題**（一般性）適合 **快速獲取建議**，但 **缺乏結構化規劃**。  
- **第二個問題**（專業性）提供 **具體可執行的30天學習計畫**，**適合有長期目標的學習者**。  

💡 **建議**：如果你的目標是 **全面提升英語口說能力**，可以參考 **第二個專業規劃**，按照每日學習計畫執行，確保穩定進步！ 🚀
